## Running Locally

To run this project localy you **need MongoDB** installed and running on your machine.

Running backend
``` bash
cd ./bff
npm install
npm run start:dev
```

Running frontend
``` bash
cd ./frontend
npm install
npm run dev
```

App will be available at [http://localhost:4000/](http://localhost:4000/)
